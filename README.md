# node-rust-starter

A starter which describe how you use rust with node.

## Run the code

```bash
git clone https://github.com/rjoydip/node-rust-starter.git
cd node-rust-starter
npm install
npm run build // build rust code
npm start
```

## Output

```js
addition of 2 and 3 >> 5
multiple of 2 and 3 >> 6
```
