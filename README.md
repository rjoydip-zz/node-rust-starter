# node-with-rust

This project tries to showcase how you can use Rust from Node.

## Run the code

```bash
git clone https://github.com/rjoydip/node-with-rust.git
cd node-with-rust
npm install
npm run build // build rust code
npm start
```

## Output

```js
addition of 2 and 3 >> 5
multiple of 2 and 3 >> 6
```
